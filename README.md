# ReText - Secondhand Book Exchange for Students

ReText is a web platform built with Flask that allows college and university students to **buy and sell used textbooks** within a trusted, verified campus environment. Students can post listings, search by course code, and save money while promoting sustainable learning.

---

## Features

- College email-based signup/login (e.g., `@myseneca.ca`)
- Post and view textbook listings
- Search by title or course code
- Personalized listings by college
- Upload cover photos
- Save/bookmark favorite listings
- Rate sellers

## Tech Stack

- **Backend**: Python, Flask
- **Frontend**: HTML, CSS (custom)
- **Database**: SQLite
